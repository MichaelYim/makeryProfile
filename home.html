jamScope

http://jamscope.in

Love making music but can't find a jam partner? Want to find more people to form a band? Try Jamscope.in! Uses a map view and search function for finding music lovers around you. Also features a real time chat function.

Tools: Meteor.js, MongoDB, GoogleMaps API, Facebook API, Javascript/Coffeescript, Underscore.JS, Handlebars.js, Twitter Bootstrap, Select2, Noty, HTML5, CSS3

Features:

  • Facebook login and picture-pull
  • Map view for locating musicians
  • Private chats (Similar to Facebook chat. Built from scratch.)
  • Search users by instrument
  • Social network features (overlay profiles, video embedding etc.)
  • More features on the way...

Developers: 1.

Kirstens Zoo

kirstenszoo.adopt.meteor.com/

This web application was a prize-winner at Code4Good Hackathon 2014. The goal was to build a website/webapp for a non-profit organization named Kirstens Zoo, with a few specific features. We built a 90% functional web-app in a 14 hour sprint. The features are listed below.

Tools: Meteor.js, MongoDB, PayPal API, GridFS, GoogleMaps API, Javascript, Underscore.JS, Handlebars.js, Twitter Bootstrap, HTML5, CSS3

Features:

  • Required: Easy image upload (for pet profiles)
  • Required: PayPal integration for donations
  • Required: Contact form
  • Sign up system for admin accounts
  • Admin interface for adding to site and viewing messages
  • Blog
  • Map view for sharing adoption stories
  • ...some features disabled due to recent bugs in dependencies

Developers: 2.

Hours:: 14.

Frontpacker

frontpacker.meteor.com

Sick of having to pack so much for your trips? Tired of carrying around so much while backpacking? Hangers, makeup, lotion, utensils... imagine one day, all of these become available on one app, and all you need to pack is your phone. Frontpacker is a platform for fellow travelers to leave behind items they don't need, so that other travelers can make good use of them.

Tools: Meteor.js, MongoDB, GoogleMaps API, Facebook API, Javascript/Coffeescript, Fontawesome.io, Less.js, Moment.js, Handlebars.js, Twitter Bootstrap, Select2, HTML5, CSS3

Features:

  • Facebook and Email Login
  • Drop pins to indicate what and where items can be found
  • Search items in list view
  • Drop chat pins to have location-based chat rooms
  • Profiles to communicate and express thanks

Developers: 1.
